Press A Key or Tap to Start

How to Play The Simon Game:

1. The game starts by showing you a colored button.

2. Click the buttons in the same order they were shown.

3. Every time you get it right, the game adds another color to remember.

4. Repeat the entire sequence correctly to move to the next level.

5. The game ends if you make a mistake.

Try to remember and repeat the longest sequence possible!
